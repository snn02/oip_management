# Организационная структура

Редакция документа по PDF: `27.01.2026`.

## 1. Классификация структурных единиц

### 1.1. Виды структурных единиц по размеру

**Отдел (подразделение)** — структурная единица, которая:

- имеет руководителя;
- отвечает за один или несколько видов бизнеса либо за одну или несколько функций бизнеса;
- может иметь вложенную структуру, состоящую из групп.

**Группа** — структурная единица, которая:

- имеет руководителя;
- отвечает за одну функцию бизнеса либо за совокупность бизнес-процессов в рамках одной функции;
- не имеет вложенной структуры.

### 1.2. Виды структурных единиц по виду деятельности

### 1.2.1. Зеленые подразделения

Зеленые подразделения реализуют функции развития направления бизнеса. Руководитель отвечает за бизнес в целом и формирование чистой прибыли.

В Севастополе к таким направлениям относятся:

- внедрение корпоративных систем 1С;
- реализация интернет-проектов;
- аутсорсинг специалистов в интересах ГК "1С-Рарус".

Дополнительно:

- зеленые подразделения могут иметь вложенную иерархию, состоящую из желтых подразделений, либо не иметь вложенной структуры;
- зеленые подразделения подчиняются РО.

### 1.2.2. Красные подразделения

Красные подразделения реализуют функции поддержки развития и сопровождения бизнеса. Руководитель подразделения отвечает за выполнение планов по целям развития, обеспечение качества процессов, а также контроль уровня затрат внутренних заказчиков.

В Севастополе красные функции делятся на:

- аккаунтинг клиентов;
- маркетинг;
- разработку продуктов и исследования;
- управление качеством.

Дополнительно:

- красные подразделения могут формировать иерархию однотипных красных подразделений;
- красные подразделения подчиняются РО.

### 1.2.3. Желтые подразделения

Желтые подразделения реализуют основную производственную функцию по соответствующему виду бизнеса.

Руководитель желтого подразделения отвечает за:

- исполнение, развитие и масштабирование производственных процессов своей специализации в рамках направления бизнеса;
- повышение производственной мощности подразделения или группы за счет выполнения плана численности и повышения уровня квалификации специалистов;
- выполнение плана по валовой прибыли, сформированного на основе производственной мощности подразделения или группы.

Дополнительно:

- желтые подразделения могут иметь вложенную иерархию;
- руководитель подразделения является играющим тренером и экспертом в своей специализации.

### 1.2.3.1. Подходы к выделению желтых производственных групп

Выделение желтых производственных групп может выполняться по следующим подходам:

1. По функциональной экспертизе — основной подход.
   Примеры:
   - по специализациям, принятым в компании и офисе;
   - разработчики 1С;
   - WEB-разработчики;
   - методологи 1С и т.д.
2. По технической экспертизе.
   Пример: группа повышения производительности, которая может включать специалистов различных специализаций.
3. По схожести и полноте реализации ключевого процесса.
   Пример: группа сопровождения, которая может включать специалистов различных специализаций.
4. По исторически сложившимся проектным группам.
   Пример: группа может быть сформирована на основе проектной команды долгосрочного проекта продолжительностью один год и более.

### 1.2.3.2. Подходы к изменению количества групп

При росте численности количество групп может увеличиваться в следующих случаях:

- в текущей группе есть специалист, готовый взять на себя роль руководителя новой группы;
- руководитель одной из групп готов взять на себя управление двумя группами с разной технической или функциональной экспертизой;
- осуществляется найм команды в полном составе;
- появляется новое направление в производстве.

Пример:

- 1С-производительность.

При росте численности количество групп может уменьшаться в следующем случае:

- при росте функциональных или технических компетенций руководителя одной из действующих групп возможно объединение двух и более групп под единым руководством, с сохранением вложенной иерархии либо без неё.
