# Список файлов проекта

- `deploy-portal.yml` — GitHub Actions workflow для публикации портала из папки `99-portal` на GitHub Pages.
- `README.md` — корневое описание репозитория со структурой разделов и ссылками на ключевые документы.
- `glossary.md` — глоссарий проекта с основными терминами, сокращениями и рабочими определениями.
- `oip-management-model-goals-okr-kpi.md` — описание управленческой модели направления через цели, OKR, KPI и мотивацию.
- `current-vision-coverage.md` — карта покрытия vision уже подготовленными стратегиями по направлениям.
- `oip-vision-2026-2028.md` — базовый документ с видением развития ОИП на 2026-2028 годы.
- `oip-sales-metrics-guide.md` — гайд по метрикам, KPI и формулам расчёта для направления продаж.
- `oip-sales-strategy-2026.md` — стратегия развития направления продаж на 2026 год.
- `oip-project-metrics-guide.md` — гайд по метрикам и показателям для проектного направления.
- `oip-project-strategy-2026.md` — стратегия развития проектного направления на 2026 год.
- `oip-support-metrics-guide.md` — гайд по метрикам и KPI для направления сопровождения.
- `oip-support-strategy-2026.md` — стратегия развития направления сопровождения на 2026 год.
- `oip-smb-metrics-guide.md` — гайд по метрикам и контрольным показателям для направления SMB.
- `oip-smb-strategy-2026.md` — стратегия развития направления SMB на 2026 год.
- `oip-delivery-metrics-guide.md` — гайд по метрикам эффективности и качества для delivery-направления.
- `oip-delivery-strategy-2026.md` — стратегия развития delivery-направления на 2026 год.
- `index.html` — главная HTML-страница портала стратегии с обзором направлений и ссылками на документы.
- `sales.html` — HTML-шаблон страницы направления продаж, заполняемый данными из JavaScript.
- `projects.html` — HTML-шаблон страницы проектного направления, заполняемый данными из JavaScript.
- `support.html` — HTML-шаблон страницы направления сопровождения, заполняемый данными из JavaScript.
- `smb.html` — HTML-шаблон страницы направления SMB, заполняемый данными из JavaScript.
- `delivery.html` — HTML-шаблон страницы delivery-направления, заполняемый данными из JavaScript.
- `styles.css` — общие стили портала, определяющие сетку, типографику, карточки и адаптивность.
- `app.js` — клиентский скрипт, который собирает и рендерит страницы портала из структуры данных.
- `data.js` — единый источник данных портала с описаниями направлений, целей, OKR, KPI и ссылками на документы.
- `interest-registration-requirements.md` — рабочий документ с требованиями к фиксации и оформлению интересов или входящих обращений.
- `company-description.md` — рабочая markdown-версия PDF с описанием компании, направлений деятельности, компетенций и ключевых показателей.
- `oip-sales-roles-profiles.md` — черновой документ по ролям, зонам ответственности и профилям в направлении продаж.
- `organizational-structure.md` — рабочее описание организационной структуры с распределением ролей и связей между функциями.
- `project-files-overview.md` — сводный список файлов проекта с кратким пояснением назначения каждого файла.
- `sales-regulations-materials.md` — подборка материалов и наработок по регламентам, связанным с направлением продаж.
