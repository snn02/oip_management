# Метрики продаж ОИП

Документ описывает систему измерения результатов работы направления продаж
Отдела интернет‑проектов (ОИП).

Документ содержит:

1. Метрики измерения Key Results (KR)
2. KPI руководителя продаж
3. Дополнительные управленческие метрики продаж

Документ используется для:

- регулярного анализа работы продаж
- подготовки управленческих отчетов
- оценки результативности руководителя продаж
- диагностики проблем в системе продаж

---------------------------------------------------------------------

# 1. Метрики измерения Key Results (KR)

KR используются для отслеживания достижения целей направления продаж.
Они описывают состояние системы продаж и динамику её развития.

---------------------------------------------------------------------

## KR1 — Регистрация интересов

**Цель:** обеспечить полный учёт входящих обращений.

Метрика:

Доля зарегистрированных обращений.

Формула:

registered_interests / total_incoming_interests

Где:

- registered_interests — количество обращений, зарегистрированных в системе
- total_incoming_interests — общее количество поступивших обращений

Целевое значение:

100%

---------------------------------------------------------------------

## KR2 — Скорость первичного контакта

**Цель:** обеспечить оперативную реакцию на входящие обращения.

Метрика:

Среднее время первичного контакта.

Формула:

Σ (время первого контакта − время поступления обращения) / количество обращений

Измеряется в часах или рабочих днях.

Целевое значение:

≤ 1 рабочий день

---------------------------------------------------------------------

## KR3 — Квалификация интересов

**Цель:** фиксировать первичную потребность клиента.

Метрика:

Доля интересов с зафиксированной квалификацией.

Формула:

qualified_interests / total_registered_interests

Где:

- qualified_interests — интересы с заполненным описанием потребности
- total_registered_interests — все зарегистрированные интересы

Целевое значение:

100%

---------------------------------------------------------------------

## KR4 — Скорость подготовки коммерческого предложения

**Цель:** ускорить цикл пресейла.

Метрика:

Среднее время подготовки коммерческого предложения.

Формула:

Σ (дата отправки КП − дата регистрации интереса) / количество предложений

Целевое значение:

≤ 5 рабочих дней

---------------------------------------------------------------------

## KR5 — Передача клиента в сопровождение

**Цель:** обеспечить завершение жизненного цикла проекта.

Метрика:

Доля проектов, по которым принято решение о дальнейшей работе.

Формула:

projects_with_decision / completed_projects

Где:

projects_with_decision — проекты, по которым зафиксировано:

- передан в сопровождение
- завершён без сопровождения

Целевое значение:

100%

---------------------------------------------------------------------

# 2. KPI руководителя продаж

KPI используются для расчёта премии и оцениваются ежеквартально.

---------------------------------------------------------------------

## KPI 1 — Валовая прибыль ОИП

Главный финансовый показатель эффективности продаж.

Формула:

Валовая прибыль = Выручка − Прямая себестоимость

Где:

- Выручка — суммарная выручка всех направлений ОИП
- Прямая себестоимость — стоимость производственных трудозатрат

Метрика премирования:

фактическая ВП / плановая ВП

Порог включения премии:

≥ 80%

---------------------------------------------------------------------

# 3. Управленческие метрики продаж

Эти метрики не используются для расчёта премии,
но регулярно анализируются руководителем продаж
и руководителем ОИП.

Они помогают диагностировать состояние системы продаж.

---------------------------------------------------------------------

## Метрика 1 — Структура потока обращений

Показывает распределение входящего потока.

Метрика:

- проекты
- сопровождение
- СМБ‑задачи
- лицензии
- конкурсы

Формула:

type_interests / total_interests

---------------------------------------------------------------------

## Метрика 2 — Конверсия интересов

Формула:

signed_contracts / qualified_interests

Используется для анализа эффективности пресейла.

---------------------------------------------------------------------

## Метрика 3 — Длина цикла сделки

Среднее время от регистрации интереса до подписания договора.

Формула:

Σ (дата договора − дата интереса) / количество сделок

---------------------------------------------------------------------

## Метрика 4 — Прогноз запуска проектов

Метрика используется для синхронизации продаж и производства.

Показывает:

- количество проектов в пресейле
- планируемые даты запуска

---------------------------------------------------------------------

## Метрика 5 — Дебиторская задолженность

Формула:

overdue_receivables / total_receivables

Используется для контроля финансовых рисков.

---------------------------------------------------------------------

# 4. Использование метрик

Метрики применяются на следующих управленческих уровнях:

Еженедельно:

- анализ входящих интересов
- скорость реакции
- статус сделок

Ежемесячно:

- анализ воронки продаж
- структура выручки
- развитие клиентов

Ежеквартально:

- расчёт KPI
- анализ валовой прибыли
- оценка результативности продаж
